Welcome to the OnnxControl created by SignalPop! 
--------------------------------------------------------------------------

Current ONNX Version Supported: 1.7

The OnnxControl is an open-source technology that gives Windows C# 
developers easy access to the  ONNX AI Model format.  

To ensure compatibility with the latest ONNX version, this project 
uses the ONNX proto files (provided by https://onnx.ai under the MIT 
License at https://github.com/onnx/onnx/blob/master/LICENSE).

Just call add the single line of code below to quickly test out
downloading, opening and then re-saving the small (26.6K) ONNX 
model used with the MNIST hand-written character database.

	OnnxSample.OnnxSample.RunSample();

------------------
For information on the ONNX data, see https://onnx.ai.

MyCaffe uses the OnnxControl to convert between MyCaffe and CAFFE 
model files formats and the ONNX Model file format.

For more information on MyCaffe, see https://www.nuget.org/packages?q=MyCaffe
or visit us on Git at www.github.com/mycaffe
